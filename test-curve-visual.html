SVG Path Explanation

M 0 18              // Move to starting point (18px down from top)
C 0 9, 9 0, 18 0    // Cubic curve: concave top-left (control points create inward curve)
L 42 0              // Line to top-right corner
L 42 74             // Line down to bottom-right corner
L 18 74             // Line to bottom-left curve start
C 9 74, 0 65, 0 56  // Cubic curve: concave bottom-left (control points create inward curve)
Z                   // Close path back to start

Key: The C (cubic Bezier) commands create the concave (inward) curves by positioning control points that pull the curve inward, creating the smooth concave shape that transitions from the sidebar to the content area.
